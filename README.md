### Instruções para rodar
Por ser um projeto realizado com **Angular**, há a necessidade do **NodeJS**. Com ele instalado basta seguir os seguintes passos.

No terminal, clone o projeto:
```
git clone 
```

Instale as dependências:
```
npm install
```

Execute a aplicação:
```
npm start 
```
